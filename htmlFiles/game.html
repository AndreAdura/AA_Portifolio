Games

These are my games built during my academic research through different tools and techniques used to develop different kind of games with the most variate concepts and purposes. In this section you can find games built in HTML, C++, C# wich were developed with the very useful help of many libraries and tutorials as SFML, CreateJS and others. In this section you can also see my game project created to my School wich is an Sandbox Game used as a plataform to expose different Portfolios from Game Designer, Graphic Designers, Interior Designers, General Programers and other artists that may be intrested in show your work in a interactive 3D enviroment.


Timberman(C++)

This is a classic game with a retro look built with the C++ language and the SFML library to generate and integrate the assets to the scene.


Real Timberman(C++)

This game has the same game logic and code that the classic Timberman but after a little work with the Sprites in the SFML and some Image Treatment I was able to switch the art-style of the game to a "Live-Action" version of Timberman.



Hazard Warehouse(Unity)

This is a game created in Unity 3D using the features of an excellent tutorial on Lynda.com where I could apply different techniques like Animation, Game Structure, Prefab Concepts, Level Design and Game Design.
